# Datasets executáveis

Casos versionados do ASEF. `case.json` usa contratos sem dependência de YAML. Referências em `generation_input_refs` podem ser fornecidas ao gerador; `oracle_ref` e `evaluation_input_refs` são materializados somente na avaliação.

O repositório é público. “Isolado” significa fora do contexto do componente avaliado, não conteúdo confidencial para o leitor.
