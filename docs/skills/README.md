# Skills ASEF

Skills são contratos versionados de Quality Engineering consumidos pelo workflow ASEF. Não são prompts livres, plugins automaticamente confiáveis ou permissões para acessar ambientes.

## Contratos

- `unit.md` — testes isolados de funções, classes e módulos;
- `backend-api.md` — automação de APIs autorizadas;
- `web-ui.md` — automação de interfaces web autorizadas;
- `mutation.md` — avaliação da sensibilidade de uma suíte existente;
- `performance.md` — workloads e thresholds previamente aprovados;
- `skill-contract-template.md` — campos obrigatórios para novas skills.

O catálogo resume a visão. A matriz de linguagens e o código do perfil declaram o suporte real.
